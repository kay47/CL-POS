Bulk Add Products

How to Bulk Add Products:
  1. Download the CSV template below
  2. Fill in your product data using the template format
  3. Upload your completed CSV file
  4. SKUs will be automatically generated based on category
  5. Review the results and fix any errors if needed
Step 1: Download Template

Download the CSV template with the correct column headers and sample data.

Download CSV Template
Step 2: Upload Your CSV File
Only CSV files are accepted. Maximum file size: 5MB
CSV Format Requirements
Column Required Format Description
name Required Text Product name
category Required Text Product category (electronics, clothing, food, etc.)
description Optional Text Product description
purchase_price Required Number Cost price (e.g., 10.50)
full_price Required Number Full pack selling price (e.g., 15.75)
quantity Optional Integer Initial stock quantity (defaults to 0)
Category Options:
  • electronics
  • clothing
  • food
  • beauty
  • books
  • home
  • sports
  • automotive
  • toys
  • health
  • jewelry
  • music
  • pets
  • office
  • tools
  • other
Important Notes:
  • SKUs are auto-generated based on category (e.g., ELE0001, CLO0001)
  • Full price must be greater than purchase price
  • Half price is automatically calculated as 50% of full price
  • Numbers should use decimal point (.) not comma
  • Category must be exactly one of the valid options listed above
  • The first row must contain column headers exactly as shown above
Sample Data Format

This is how your CSV should look:

name category description purchase_price full_price quantity
Premium Rice 5kg food High quality jasmine rice 25.00 35.00 50
Smartphone Case electronics Protective phone case 8.00 15.00 25
Cotton T-Shirt clothing 100% cotton casual wear 12.00 20.00 30

Result: These will get SKUs like FOO0001, ELE0001, CLO0001 automatically!
