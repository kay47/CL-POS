Step 2: Upload Your CSV File
Only CSV files are accepted. Maximum file size: 5MB
CSV Format Requirements
Column Required Format Description
sku Required Text Unique product SKU/code
name Required Text Product name
description Optional Text Product description
purchase_price Required Number Cost price (e.g., 10.50)
full_price Required Number Full pack selling price (e.g., 15.75)
quantity Optional Integer Initial stock quantity (defaults to 0)
Important Notes:
  • Each SKU must be unique - duplicate SKUs will be skipped
  • Full price must be greater than purchase price
  • Half and quarter pack prices are calculated automatically
  • Numbers should use decimal point (.) not comma
  • The first row must contain column headers exactly as shown above
